## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## PWA Features

- **Installable**: Can be installed on devices like a native app
- **Offline Support**: Works offline with service worker caching
- **Responsive**: Adapts to different screen sizes
- **Fast Loading**: Optimized with Vite bundling

## Icons

Replace the placeholder icon files in `/public/` with actual PNG images:
- `pwa-192x192.png` (192x192 pixels)
- `pwa-512x512.png` (512x512 pixels)

## Testing PWA

1. Build the app: `npm run build`
2. Serve it: `npm run preview`
3. Open in Chrome and check the install prompt
4. Test offline functionality in DevTools
